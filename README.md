# Hunger Games Countdown

Каунтдаун до ревью в стиле "Голодных игр"

## Описание

HTML страница с драматичным каунтдауном до даты ревью (15.06.2026 13:00 МСК) в стиле неба из фильма "Голодные игры".

## Функции

- Фоновое изображение неба из "Голодных игр"
- Автоматическая смена фотографий участников команды каждые 5 секунд
- Анимированный каунтдаун с падающими цифрами (дни, часы, минуты)
- Мгновенное обновление секунд
- Драматичные эффекты и цветовая схема

## Использование

Откройте файл `hunger_games_countdown.html` в браузере.

## GitHub Pages

Для публикации на GitHub Pages:
1. Загрузите файл в репозиторий
2. Перейдите в Settings → Pages
3. Выберите источник: Deploy from a branch → main
4. Страница будет доступна по адресу: https://mdanilyuk.github.io/until_review/hunger_games_countdown.html
